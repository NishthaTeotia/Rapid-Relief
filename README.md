# Rapid- Relief
 RapidRelief is a full-stack disaster response web application designed to help communities report incidents, coordinate volunteers, and manage emergency response activities in real-time. The platform enables users to quickly submit disaster reports, track affected locations on an interactive map, and assign tasks efficiently during emergencies.

🚀 Features

🔐 Authentication & Authorization
Secure JWT-based authentication
Role-based access control (Admin, Volunteer, User)
Protected routes and APIs

📍 Incident Reporting
Users can report disasters
Add incident details 

🗺️ Map Integration
Interactive maps for location tracking
External map APIs integration for geolocation services

👥 Volunteer Coordination
Volunteers can view assigned tasks
Admins can manage and assign relief tasks
Status tracking for ongoing operations

🔄 RESTful API System
CRUD operations for incidents, users, and tasks
Structured backend APIs using Express.js
Efficient MongoDB data management

🛠️ Tech Stack

Frontend

CSS 

Backend
Node.js
Express.js

Database
MongoDB

Authentication
JSON Web Token (JWT)
